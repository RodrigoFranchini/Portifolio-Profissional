## 🛠️ Como rodar localmente

### Pré-requisitos
- **Node.js - inclui o npm**

### Passos
```bash
# 1) Instalar dependências
npm install

# 2) Rodar em modo desenvolvimento
npm run dev
# O Vite mostrará a URL local, algo como: http://localhost:5173/

```
